    このgitリポジトリについて¶

    nuxt3を用いてwebアプリケーションを構築してサーバーレスデプロイするためのベースとするもの

    基本方針¶

    • コンテナ環境において開発作業する
      • SPA構成を取りSSRしない
      • Nitroでbuildする
    • slsを用いてサーバーレスデプロイできる
      • aws lambdaで関数URL化する
      • api gatewayにエンドポイントを立ててアクセスできる
    • その他、開発に有用なツールをいくつか積んでおくこと

    nuxt3 フルスタックwebフレームワーク
    vuetify マテリアルデザインコンポーネント
    i18n 多言語対応
    pug テンプレートエンジン
    typescript 静的型付け
    prisma orm
    eslint 静的解析ツール
    prettier フォーマッター
    vitest テスト
